# Customer Shopping Behaviour Analysis(End to end) using (ML + NLP)
Understanding customer shopping behaviour is crucial for improving sales, customer retention, and personalization. This project analyses customer data using Machine Learning (ML) and Natural Language Processing (NLP) to uncover patterns and predict customer actions such as subscription decisions.

# This project is perfect for:

•	Perform Exploratory Data Analysis (EDA) on customer data 

•	Identify patterns in purchasing behaviour

•	Analyze customer reviews using sentiment analysis (NLP)

•	Build a Machine Learning model to predict subscription status

•	Generate actionable business insights

# NLP – Sentiment Analysis

# Customer reviews were analyzed using NLP techniques:

# Steps:

1.	Text cleaning (lowercase, remove stopwords)
   
2.	Sentiment scoring using polarity
   
3.	Classification into:
   
o	Positive 

o	Negative

o	Neutral 

# Insight:

•	Positive sentiment customers are more likely to subscribe

•	Negative sentiment highlights product/service issues
